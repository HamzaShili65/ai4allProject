## About The Project
Two different forms of analysis and models used to predict the likelihood of homelessness for women and children, and possibly preventing it.
<br><br>
Linear regression is used to analyze homelessness and other factors like income levels, demographics, and housing policies.

K-means clustering is used to find patterns and groupings in populations or regions based on numerous factors.
